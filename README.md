# Баланс работы и отдыха (расширение VSCode)

Лёгкое расширение, которое отслеживает время работы в строке состояния и напоминает о необходимости размяться или сделать упражнения для глаз.

## Быстрая установка (для разработчиков / локально)

1. Запустите скрипт-генератор, который создал эту папку.  
2. Скопируйте папку `work-life-balance` в каталог расширений VSCode:

   ```sh
   # для macOS
   cp -R work-life-balance ~/.vscode/extensions/work-life-balance
   ```

3. Перезапустите VSCode.

Альтернативно: откройте папку в VSCode и нажмите F5, чтобы запустить Extension Development Host для тестирования.

## Команды

- **Work-Life: Open Exercises** — открывает панель с упражнениями и таймерами для короткого перерыва.
- **Work-Life: Start/Stop Timer** — запускает или останавливает внутренний таймер (в строке состояния отображается время работы).
- **Work-Life: Reset Timer** — сбрасывает накопленное время.

## Настройки (Настройки → Расширения → Work-Life Balance)

- `workLifeBalance.remindMinutes` — минуты между напоминаниями о разминке (по умолчанию 50).
- `workLifeBalance.eyeExerciseMinutes` — минуты между напоминаниями о правиле 20-20-20 для глаз (по умолчанию 20).
- `workLifeBalance.snoozeMinutes` — время откладывания напоминания (по умолчанию 10).
